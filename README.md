GeoWeb3 - Децентрализованная Карта Отзывов

Это пошаговая инструкция по запуску вашего Web3-приложения.

Шаг 1: Развертывание Смарт-Контракта (GeoReviews.sol)

Ваш смарт-контракт — это ваш "бэкенд". Его нужно опубликовать (развернуть) в тестовой сети Sepolia. Мы будем использовать Remix IDE, популярный онлайн-инструмент для этого.

Откройте Remix: Перейдите на remix.ethereum.org.

Создайте файл: В левой панели создайте новый файл с именем GeoReviews.sol.

Скопируйте код: Вставьте в этот файл полное содержимое файла GeoReviews.sol, который я предоставил.

Скомпилируйте контракт:

Перейдите на третью вкладку слева (Solidity Compiler).

Убедитесь, что версия компилятора (Compiler) установлена на 0.8.20 (или совместимую, например, 0.8.24).

Нажмите большую синюю кнопку "Compile GeoReviews.sol".

Если все в порядке, вы увидите зеленую галочку.

Скопируйте ABI:

Сразу после компиляции на той же вкладке появится кнопка "ABI".

Нажмите на нее, чтобы скопировать ABI в буфер обмена.

Вставьте этот ABI в Блокнот. Он понадобится вам на Шаге 2. Это длинный JSON-массив.

Разверните контракт:

Перейдите на четвертую вкладку слева (Deploy & Run Transactions).

В выпадающем списке "ENVIRONMENT" выберите "Injected Provider - MetaMask".

Ваш MetaMask должен всплыть и запросить подключение. Подключитесь.

Убедитесь, что ваш MetaMask переключен на сеть Sepolia. У вас должны быть тестовые ETH (вы упомянули, что они есть).

В поле "CONTRACT" убедитесь, что выбран GeoReviews.

Нажмите оранжевую кнопку "Deploy".

MetaMask снова всплывет и попросит вас подтвердить транзакцию (и оплатить "газ"). Подтвердите.

Получите адрес контракта:

Подождите 15-30 секунд. После успешного развертывания внизу, в секции "Deployed Contracts", появится ваш контракт.

Нажмите на иконку "копировать" рядом с адресом вашего контракта (например, 0xAbC...123).

Сохраните этот адрес. Он понадобится вам на Шаге 2.

Шаг 2: Настройка Фронтенда (index.html)

Теперь вам нужно "связать" ваш сайт с вашим только что развернутым контрактом.

Откройте index.html: Откройте файл index.html в любом текстовом редакторе (VS Code, Блокнот и т.д.).

Найдите секцию настроек: Прокрутите вниз до тега <script type="module"> (почти в самом низу файла). Вы увидите этот блок:

// -----------------------------------------------------------------
// !! ВАЖНО: ШАГ 1 - Настройте эти переменные !!
// -----------------------------------------------------------------

const CONTRACT_ADDRESS = "0x...ВАШ_АДРЕС_КОНТРАКТА";

const CONTRACT_ABI = [
    // ... Пример ABI ...
];
// -----------------------------------------------------------------


Вставьте Адрес: Замените "0x...ВАШ_АДРЕС_КОНТРАКТА" на адрес, который вы скопировали на Шаге 1 (пункт 7).

Вставьте ABI: Полностью замените весь массив [...] (включая квадратные скобки) на ABI, который вы скопировали на Шаге 1 (пункт 5).

Сохраните файл index.html.

Шаг 3: Публикация на GitHub Pages

Создайте репозиторий: На GitHub создайте новый публичный репозиторий (например, my-geo-dapp).

Загрузите файл: Загрузите ваш отредактированный index.html в этот репозиторий.

Включите Pages:

В вашем репозитории перейдите в Settings (Настройки).

В боковом меню выберите Pages.

В разделе "Branch" выберите main (или master) и папку /root.

Нажмите Save.

Готово! Через 1-2 минуты ваш сайт будет доступен по адресу https://<ВАШ_НИКНЕЙМ>.github.io/<ИМЯ_РЕПОЗИТОРИЯ>/.

Теперь вы можете открыть этот URL, подключить MetaMask (убедитесь, что он в сети Sepolia) и начать оставлять отзывы, которые будут сохраняться прямо в блокчейне!
